# MbokaPay API Documentation

Bienvenue sur la documentation API MbokaPay.

## Base URL

- Sandbox: [https://staging.mbokapay.com](https://staging.mbokapay.com)

- Live: [https://mbokapay.com](https://mbokapay.com)

## Authentication

Header required: X-API-Key

## Endpoints

- GET /api/v1/health

- POST /api/v1/cardholders

- POST /api/v1/cards

- GET /api/v1/cards/{id}

- GET /api/v1/api-keys

- POST /api/v1/api-keys

- DELETE /api/v1/api-keys/{id}

- GET /api/v1/webhooks

- POST /api/v1/webhooks

- DELETE /api/v1/webhooks/{id}

- GET /api/v1/user

- GET /api/v1/user/transactions

- POST /api/v1/user/device-token
